## About

Lovely-Admin-Dashboard is a clean, visually pleasing admin panel template.  
It moves away from cluttered, boring backends and brings soft colors, rounded corners, and smooth interactions.

Perfect for:
- SaaS dashboards
- CMS admin panels
- Internal tools
- Personal projects

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Lomi-1127/Lovely-Admin-Dashboard.git

# Navigate into the folder
cd Lovely-Admin-Dashboard

# Install dependencies
npm install

# Start development server
npm run dev
